LiveAtom

A lightweight runtime CSS library inspired by Tailwind. LiveAtom dynamically parses utility classes and injects optimized stylesheets directly into the CSSOM on the fly.

Get Started View Utilities

Why Choose LiveAtom?

01.

Zero-Build Runtime

Skip Webpack, Vite, or PostCSS configurations. Just drop the script into your HTML and start styling immediately in the browser.

02.

Direct CSSOM Injection

Bypasses heavy DOM style blocks by writing parsed rules directly into the browser's CSS Object Model for maximum performance.

03.

Live DOM Watching

Powered by native MutationObservers. If you inject HTML components via JavaScript later, LiveAtom computes and applies styles instantly.

04.

Smart Style Caching

Guarantees high-speed execution after the initial parse by caching unique classes and completely preventing redundant CSS rule generation.

05.

Static & Dynamic Utilities

Access hundreds of pre-mapped utilities or use Just-In-Time arbitrary values like atom-w-[250px] on the fly.

06.

Baseline Style Reset

Includes a modern, automated CSS reset injected by default, ensuring absolute cross-browser consistency without an extra stylesheet.

07.

Extensible Custom API

Need a specialized class? Register custom CSS blocks instantly directly in your JavaScript using the liveAtom.addClass() method.

08.

Framework Agnostic

Seamlessly integrates anywhere. Use it with plain HTML/Vanilla JS, modern frameworks like React and Vue, or backend templates like PHP.

Installation & Usage

Via NPM

1. Install the package

npm install live-atom

2. Set up the Import Map

<script type="importmap"> { "imports": { "live-atom": "./node_modules/live-atom/main.js" } } </script>

3. Import as a Module

import liveAtom from "live-atom";

Via CDN

<script src="https://unpkg.com/live-atom"></script>

## 💡 What is LiveAtom?

**LiveAtom** is a revolutionary utility-first runtime CSS library. Inspired by the Tailwind, it eliminates the need for bulky build tools, continuous watchers, or massive static `.css` exports.

Instead, LiveAtom **dynamically parses your HTML for utility classes at runtime** and injects highly optimized, deduplicated styles directly into the browser's CSS Object Model (CSSOM) on the fly.

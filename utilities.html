Pseudo-Classes (Hover, Focus, etc.)

The Atom engine automatically parses pseudo-class prefixes. You can prepend any utility class with state modifiers.

Supported States:

  • hover: - Applies on mouse hover.
  • focus: - Applies when element has DOM focus.
  • active: - Applies when element is being clicked.
  • disabled: - Applies when a form element is disabled.

Live Example:

Classes used: atom-bg-blue-100 hover:atom-bg-blue-600 hover:atom-text-white focus:atom-border focus:atom-border-black

Extending the Engine (Custom Classes)

You can dynamically inject custom component classes into the engine at runtime using the atomLive.add() API.

⚠️ Important Constraint: All custom classes registered via this API must start with the atom- prefix to be processed by the engine. 
// Register a custom button component
atomLive.add("atom-btn-primary", {
    "background-color": "#155dfc",
    "color": "#ffffff",
    "padding": "12px 24px",
    "border-radius": "8px",
    "font-weight": "600",
    "cursor": "pointer"
});

// Now you can use class="atom-btn-primary" anywhere in your HTML!
```
